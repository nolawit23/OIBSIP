#  Iris Flower Classification (OIBSIP Task 1)

##  Objective
To build a machine learning model that classifies iris flowers into three species:
- Setosa
- Versicolor
- Virginica  
based on their sepal and petal features.

## Steps Involved
1. Data Collection and Preprocessing  
2. Feature and Label Separation  
3. Data Splitting (Training & Testing)  
4. Model Training (Random Forest Classifier)  
5. Model Evaluation (Accuracy, Confusion Matrix, Report)  
6. Hyperparameter Tuning (GridSearchCV)  
 
##  Result
- **Final Accuracy:** 93.33%
- **Best Parameters:**  
  `{ 'max_depth': 3, 'min_samples_leaf': 1, 'min_samples_split': 2, 'n_estimators': 50 }`

##  Tools Used
- Python (Jupyter Notebook)
- Pandas, NumPy
- Scikit-learn
- Matplotlib & Seaborn
